CrytoRC4.exe dosyası açıldığında program çalışacak.
Sayfaya hoşgeldiniz, RUN butonuna bastığınızda encrypt ve decrypt sayfaları çalışacak.
TextBox'a şifrelenmesini istediğiniz cümle veya kelimeyi yazabilirsiniz. (MAX 32767 Karakter)
Encyption Key şifreleme anahtarınızı yazmanız içindir.
Toplamda 4buton bulunmaktadır.
[DeCrypt] şifrelenmiş mesajı çözmek için,
[Clear] tüm TextBoxları silmek için,
[Close] uygulamayı kapatmak içindir.
